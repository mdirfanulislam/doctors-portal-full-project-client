# Features
It is a doctor’s chamber website. Patients get appointments by the date.

Patients and doctors can log in to this website using Google or an email
password.

A Patient shows all the services and books an appointment.

A doctor can see all appointments of the date booked by the patient,
show all patients, and add a doctor

Technologies Used:
● Front-End Technology: HTML, CSS, React JS, Material-Ui, React Router,
Firebase, React-Fontawesome
● Back-End Technology: Node.js, Express.js, MongoDB
